TG QR Code Generator

TG QR Code Generator is a simple Android application that allows users to generate QR codes instantly. The app is designed to be fast, lightweight, and easy to use, making QR code creation accessible for everyone.

📱 About the Project

This project focuses on providing a clean and efficient way to generate QR codes for different types of data such as text and URLs. It is built for Android and can be expanded with more features in the future.

🚀 Features
Generate QR codes instantly
Supports text and URL input
Simple and user-friendly interface
Fast performance with minimal steps
Lightweight Android application
🛠️ Built With
Android Studio
Java (or Kotlin — update if needed)
QR code generation library (e.g., ZXing)
